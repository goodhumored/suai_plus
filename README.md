### Ручная сборка
1. Склонируйте репозиторий
	```sh
	git clone https://github.com/goodhumored/suai_plus.git
	```
2. Установите необходимые пакеты и зависимости
	```sh
	npm i
	```
3. Соберите расширение
   ```sh
	npm run build
	```
Собранное расширение будет лежать в папке dist.
